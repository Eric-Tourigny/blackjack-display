# Blackjack Display

The code for an interactive blackjack display, which was designed, coded, wired, 3D printered, and assembled by 4-person team ([Anders Bouck](https://www.linkedin.com/in/anders-bouck-b9346b198/), [Chester Agustin](https://www.linkedin.com/in/chester-agustin/), [David Bughiu](https://www.linkedin.com/in/david-bughiu-809ab4323/), [Eric Tourigny](https://www.linkedin.com/in/eric-tourigny/)), for an ENGG 200 group project at the [Schulich School of Engineering](https://schulich.ucalgary.ca/). Utilizes MicroPython on a Raspberry PI Pico to control a button, LCD and LEDs.
